# Features

Guac comes with a user-friendly customizable ui for updating settings and configuration. The menus
and emulators all have controller support, and all emulators come with integer scaling, and fps options. 
The dmg / cgb emulator comes with customizable palettes, and palette presets.
The Nintendo DS comes with customizable screen layouts, rotations and sizing, real-time clock options, 3d scene exporter, and  bios / firmware.

# Installation / Building

See Releases for Windows and Linux precompiled binaries.

Building from source is possible with golang > 1.26.2, using:

```
go build .
```

# Getting Started

Run the executable without any flags or configuration.

```
.\guac
```

If you are interested in specific flags and configuration options please see [guacemulator.com](https://guacemulator.com/).
