# Gameboy Emulator


## Goals

DMG and CGB support.

- DMG
- CGB
- CGB (DMG Support)
- Cartridge ROM ONLY
- Cartridge MBC1
- Cartridge MBC2
- Cartridge MBC3
- Cartridge MBC5
- Cartridge MBC6
- Cartridge MBC7

### Unsupported
- SGB Super Gameboy Adapter
- Gameboy Printer
- Gameboy Camera
- Gameboy 4-Player Adapter
- Game Genie / Shark Cheats
- Infrared Communication
- Cartridge MMM01
- Cartridge M161
- Cartridge HuC1
- Cartridge HuC3
- Cartridge MMM01
